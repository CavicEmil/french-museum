## CSS & Design
# tailwindCSS: colours & font styles
Use App.css theme section to make changes/additions to the styling: colours & font styles
DO NOT EVER EVER EVER:
- make changes using CSS inline code
- create CSS classes (highly likely tailwindcss already has one for that)
- overwrite existing classes without checking with Emil first 
# daisyUI
We are using daisyUI to make easily reusable components stylings: button, card, title, bodytext usv.
documentation for how to use:
https://daisyui.com/docs/intro/
if you have any questions Emil is available during the usual office hours

## BACKEND/SERVER
To ensure the web app works make sure to get into the backend directory, using the command line prompt/terminal type:
"npm start"
This is going to start the server service and seed the database.
Visit http://localhost:8080/api/exhibits
to see the data.
